# AlexBelog Game Request v4

Статический сайт под GitHub Pages с публичной очередью заявок и отдельным разделом пройденных игр.

## Что добавлено в v4
- отдельный публичный раздел **Пройденные игры**
- рейтинг от **1 до 10** в виде 10 звёзд
- добавление/редактирование пройденных игр только через админку
- отдельная таблица `completed_games` в Supabase
- best-effort автоподтягивание постера из HLTB при добавлении пройденной игры
- fallback без постера, если HLTB не вернул изображение

## Файлы
- `index.html`
- `styles.css`
- `app.js`
- `supabase.sql`

## Настройка
1. Создай проект в Supabase.
2. Выполни SQL из `supabase.sql`.
3. Вставь `SUPABASE_URL` и `SUPABASE_ANON_KEY` в `app.js`.
4. Залей файлы в GitHub-репозиторий.
5. Включи GitHub Pages.
6. Добавь URL сайта в Supabase Auth Redirect URLs.

## Важно
Интеграция с HLTB для постеров работает в best-effort режиме через сторонний браузерный клиент. Если HLTB меняет поиск или поля изображений, постер может не подтянуться автоматически — в этом случае игра всё равно сохраняется, а ссылку на постер можно подставить позже вручную.
